Siap pakai data nyata?

Ikuti 3 langkah berikut untuk menghubungkan FinanceBook ke Google Sheets bisnis kamu:

1
Buat Google Sheet baru — beri nama seperti "Akuntansi PT Sinar Jaya 2025". Klik di sini →
2
Salin URL Sheet dari browser. URL terlihat seperti:
docs.google.com/spreadsheets/d/SHEET_ID/edit
3
Buat API Key di Google Cloud Console: buat project → aktifkan Google Sheets API → Credentials → Create API Key.
4
Paste URL Sheet dan API Key di tab "Hubungkan". FinanceBook otomatis membuat semua lembar yang diperlukan.
